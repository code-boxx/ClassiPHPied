## CLASSIPHPIED
ClassiPHPied is a simple PHP Classified Ads System. This is not a "we have a load of features, but you won't even use half of them" kind of system. Just a basic one, with the essentials to help you get started quickly â€“ Be it for a small business, or expanding on top of it.
<br><br>

## FEATURES
1) Users management.
2) Ads management.
3) Progressive web app.

## REQUIREMENTS
1) LAMP/WAMP/MAMP/XAMPP
2) Apache Mod Rewrite
3) PHP MYSQL PDO Extension
4) At least PHP 8.0
<br><br>

## INSTALLATION
Just access `http://your-site.com/` in your browser and walk through the installer. The admin panel can be accessed from `http://your-site.com/admin`.
